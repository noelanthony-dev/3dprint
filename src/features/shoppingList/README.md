# Shopping List

Manual shopping list and procurement suggestion module.

Implemented scope:

- Saves manual shopping list items locally.
- Tracks category, quantity, unit, priority, status, source note, and notes.
- Generates explainable suggestions from low-stock add-ons.
- Generates explainable suggestions from missing HueForge filament requirements.
- Allows adding generated suggestions to the manual list explicitly.

Non-goals for this module:

- No online ordering.
- No external supplier APIs.
- No automatic purchasing.
- No reminders or sync.
- Generated suggestions do not change inventory.
